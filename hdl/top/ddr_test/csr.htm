  • FMC1_PRES [read-only]: FMC slot 1 presence
    0: FMC slot 1 is populated
    1: FMC 1 slot is not populated.
  • FMC2_PRES [read-only]: FMC slot 2 presence
    0: FMC slot 2 is populated
    1: FMC 2 slot is not populated.
  • SYS_PLL_LCK [read-only]: System clock PLL status
    0: not locked
    1: locked.
  • DDR3_BANK4_CAL_DONE [read-only]: DDR3 bank4 calibration status
    0: not done
    1: done.
  • DDR3_BANK5_CAL_DONE [read-only]: DDR3 bank5 calibration status
    0: not done
    1: done.
  • GPIO_IN [read-only]: GPIO inputs
    GPIO inputs value
  • RESERVED [read-only]: Reserved
    Ignore on read, write with 0's.
